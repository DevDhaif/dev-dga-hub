# Security policy

Only the latest `@dev-dga/*` versions on npm get security fixes.

Do not open a public issue for security problems. Report privately:

- GitHub: [report a vulnerability](https://github.com/DevDhaif/dev-dga-hub/security/advisories/new)
- Email: devdhaif@gmail.com with "dev-dga security" in the subject

Include what is affected, steps to reproduce, and the impact. You will get a
reply within a few days. Please allow time for a fix before public disclosure.
